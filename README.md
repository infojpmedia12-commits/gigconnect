# GigConnect

Kenya's Premier Freelance Services Marketplace with M-Pesa Integration

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Clerk Authentication
- M-Pesa Daraja API

## Getting Started

```bash
git clone https://github.com/infojpmedia12-commits/gigconnect.git
cd gigconnect
npm install
npm run dev
```
